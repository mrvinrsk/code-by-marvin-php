How does it work?

Actually, it is pretty simple! (Otherwise you wouldn't be here)

To create a scroll-element you just need a normal HTML element with the class scroll.

<p class="scroll">Text here</p>

And to that, you can just add a data-attribute, called target. So you have:

<p class="scroll" data-target="#hello">Text here</p>

As you might notice, the value of the target attribute is #hello. This means, that, if you click on it, you'll scroll to the element with the id hello

There are a few more options to get explored:

Option (data-target="...") Explanation
+section The next element of type section, starting from the element being clicked on.
-.hello The closest previous element with the class hello, starting from the element being clicked on.
span The first span element on the page
