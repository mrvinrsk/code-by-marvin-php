Diese Popups sind einfach zu verwenden. Einfach die jeweiligen Code-Schnipsel kopieren und in die eigene Webseite einfügen und anpassen. Die einzelnen Popups werden vom Code über die ID unterschieden. Du kannst völlig unkompliziert einen Button erstellen, der ein Popup öffnet. Dazu musst du nur einem beliebigen Element das Attribut data-toggle-popup mit der ID des Popups (ohne #) geben. Um was für ein Element es sich dabei handelt, ist egal. Es kann ein Button, ein Link oder ein beliebiges anderes Element sein.
