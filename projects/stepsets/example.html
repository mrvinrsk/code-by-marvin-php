Quick and Dirty

The Markup of a step-set is pretty simple actually! 

<div class="step-set">
  <div class="step NAME">
    <h2>Content here</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  </div>
</div>

Where NAME is the name for this step. More on that in the complex documentation.

Of course you have to link the stylesheet and the JavaScript code in the <head> before. Here's how you can do that: 

<link rel="stylesheet" href="stepsets.css">
<script src="stepsets.js"></script>

Please be aware, that these step-sets need jQuery in order to be fully functional.

When all of this is ready and set-up you just have to fill in your contents. To change the current step you just have to add e.g. a button with the attribute data-step-next or data-step-next, for more specific changes you may use the data-step attribute, which is explained more detailed in the complex documentation.

More of a deep-dive

Firstly, to get started, we need our markup in HTML, which happens to be really self-explanatory: 

<div class="step-set">
  <div class="step NAME">
    <h2>Content here</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  </div>
</div>

You may notice the NAME in this snippet of code. This will be explained further down, for now remember that this will be the name of this step.

As you would with any other code-snippet, you logically have to link the needed files for this snippet in your <head>, too. 

<link rel="stylesheet" href="stepsets.css">
<script src="stepsets.js"></script>

Please be aware, that these step-sets need jQuery in order to be fully functional.

When all of this is ready and set-up you just have to fill in your contents. To change the current step you just have to add e.g. a button with the attribute data-step-next or data-step-next, for more specific changes you may use the data-step attribute, which is explained more detailed here. Important notice: The data-step-next, data-step-next and data-step attributes will only work, if you're using them inside of a step-set. If you want to place them outside of it, you have to add data-step-set="NAME" additionally, so the button knows which step-set it belongs to, if you don't do this, the button will navigate the closest step-set from its position in the DOM.

More information

Navigating

It's pretty simple to change the currently active step of a step-set. There are two ways to do that:

  • Through JavaScript by using the stepset_set_active(stepset, step) function, which can be found in the JS-file. Important is, that the function doesn't take the NAME of the wanted step, as an argument for step, instead it needs the HTML-Element of the step itself. So I would recommend to only use this variant in external JS-files.
  • Through data-attributes: There are three data-attributes, which can be used:
    • data-previous-step: Used to change to the previous step in the DOM.
    • data-next-step: Is used to change to the next step in the DOM, and
    • data-step="NAME": Can be used to change to a specific step by specifying its name.
